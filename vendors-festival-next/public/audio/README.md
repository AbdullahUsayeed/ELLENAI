# Drop your audio files here

## Required files

| File | Description | Suggested source |
|---|---|---|
| `lofi-piano.mp3` | Layer 1 — soft looping lo-fi / Ghibli-esque piano (2–4 min) | [Pixabay "lofi"](https://pixabay.com/music/search/lofi/) |
| `ambient-bazaar.mp3` | Layer 2 — crickets, gentle breeze, or faint bazaar ambience | [Pixabay "nature ambience"](https://pixabay.com/sound-effects/search/night%20ambience/) |

Layer 3 (tent hover shimmer) is synthesised in-browser via the Web Audio API — no file needed.

## Tips
- Keep each file under 3 MB for fast initial load.
- Encode at 128 kbps stereo MP3.
- Both files loop, so trim them to a clean loop point.
